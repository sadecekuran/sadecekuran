Uygulamayı Cihazınıza Yükleyin

SADECE KURAN uygulamasını cihazınıza yükleyerek offline erişim sağlayın ve daha hızlı bir deneyim yaşayın.

Hızlı Kurulum

Tarayıcınız destekliyorsa, aşağıdaki butona tıklayarak uygulamayı doğrudan yükleyebilirsiniz:

Not: Eğer yukarıdaki buton görünmüyorsa, aşağıdaki manuel kurulum adımlarını takip edin.

Mobil Cihazlar

iPhone / iPad

Safari tarayıcısı kullanarak

  1. Safari'de Siteyi Açın

    SADECE KURAN web sitesini Safari tarayıcısında açın.

    📱 Safari'de site açık

  2. Paylaş Butonuna Tıklayın

    Ekranın altındaki paylaş butonuna (⬆️) tıklayın.

    ⬆️ Paylaş butonu

  3. "Ana Ekrana Ekle" Seçin

    Açılan menüden "Ana Ekrana Ekle" seçeneğini bulun ve tıklayın.

    ➕ Ana Ekrana Ekle seçeneği

  4. Uygulamayı Adlandırın

    Uygulama adını onaylayın ve "Ekle" butonuna tıklayın.

    ✅ "Ekle" butonu
Artık ana ekranınızda SADECE KURAN uygulaması var!

Android

Chrome veya diğer tarayıcılar

  1. Chrome'da Siteyi Açın

    SADECE KURAN web sitesini Chrome tarayıcısında açın.

    🌐 Chrome'da site açık

  2. Menü Butonuna Tıklayın

    Sağ üst köşedeki üç nokta (⋮) menü butonuna tıklayın.

    ⋮ Menü butonu

  3. "Ana ekrana ekle" Seçin

    Açılan menüden "Ana ekrana ekle" veya "Uygulama yükle" seçeneğini tıklayın.

    📱 Ana ekrana ekle seçeneği

  4. Kurulumu Onaylayın

    Çıkan onay penceresinde "Yükle" veya "Ekle" butonuna tıklayın.

    ✅ Kurulum onayı
Uygulama ana ekranınıza ve uygulama çekmecenize eklendi!

Masaüstü Bilgisayarlar

Google Chrome

Windows, Mac, Linux

  1. Chrome'da Siteyi Açın

    SADECE KURAN web sitesini Chrome tarayıcısında açın.

  2. Yükleme İkonunu Arayın

    Adres çubuğunun sağında yükleme ikonu (⬇️) görünecektir.

    ⬇️ Adres çubuğunda yükleme ikonu

  3. Yükleme İkonuna Tıklayın

    Yükleme ikonuna tıklayın veya menüden "Uygulama yükle" seçeneğini seçin.

  4. Kurulumu Onaylayın

    Çıkan onay penceresinde "Yükle" butonuna tıklayın.

Microsoft Edge

Windows, Mac

  1. Edge'de Siteyi Açın

    SADECE KURAN web sitesini Microsoft Edge tarayıcısında açın.

  2. Menü Butonuna Tıklayın

    Sağ üst köşedeki üç nokta (...) menü butonuna tıklayın.

  3. "Uygulamalar" Menüsüne Gidin

    Menüden "Uygulamalar" seçeneğini bulun ve tıklayın.

  4. "Bu siteyi uygulama olarak yükle" Seçin

    Alt menüden "Bu siteyi uygulama olarak yükle" seçeneğini tıklayın.
